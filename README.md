## :memo: アプリ紹介
Favorite store は、おすすめのお店を探すことができます！
- ユーザーは、次に予約するお店が見つかります
- ユーザーは、お店のブックマークリストを作ることができます
- もし行ったことがあるお店があるならば、ぜひコメントしてみてください。
